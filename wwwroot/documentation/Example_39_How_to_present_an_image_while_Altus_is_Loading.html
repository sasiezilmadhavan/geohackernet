Example #39 - How to Present an Image while Altus is Loading
This example demonstrates how to show an image while the BA3 Altus Mapping Engine is loading. Compare this example to example #2. The two differences are: 1) The img tag added to the div in the HTML section, and 2) The .centered part in CSS section. When you run this, the selected image will display for a few seconds while the BA3 Altus mapping engine is loading.
